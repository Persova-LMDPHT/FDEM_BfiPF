#### This folder contains an example of the ready-made folder for calculation with already built programs and required *.dll files in 'Modules' folder

1. Put all prepared input text files (gen, layers, objects, recE, recB, settings.cfg) in this folder.   
You can take already prepared input files from ['CalculationExample'](https://github.com/Persova-LMDPHT/FDEM_BfiPF/tree/89293a537e8dbebc5c7704a6e67aa143482e42f4/CalculationExample).
2. Run CalcStarter.exe and wait for it to complete.
3. Look for result file (e2d.1, e3d.1) in 'Results' folder inside this folder.
